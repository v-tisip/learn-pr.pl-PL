Microsoft Azure to stale rozbudowywany zestaw usług w chmurze, który pomaga organizacjom sprostać bieżącym i przyszłym wyzwaniom biznesowym. Platforma Azure umożliwia elastyczne tworzenie i wdrażanie aplikacji oraz zarządzanie nimi w dużej sieci globalnej przy użyciu ulubionych narzędzi i struktur. Ta sekcja zawiera ogólne omówienie zasobów udostępnianych na platformie Azure.

### <a name="compute"></a>Wystąpienia obliczeniowe

Firmy wybierają platformę Azure przede wszystkim ze względu na oferowane usługi obliczeniowe. Platforma Azure udostępnia szereg opcji hostowania aplikacji i usług, m.in. następujące:

- Infrastruktura jako usługa (**IaaS**)
- Platforma jako usługa (**PaaS**)
- Funkcje jako usługa (**FaaS**)

Poniżej przedstawiono udostępniane usługi:

|  Nazwa usługi             | Funkcja usługi                                                          |  Typ     |
| -------------             | -------------                                                             | --------- |
| Maszyny wirtualne          | Maszyny wirtualne z systemem Windows lub Linux hostowane na platformie Azure                                      | IaaS      |
| Azure Container Service   | Umożliwia zarządzanie klastrem maszyn wirtualnych, na których są uruchomione usługi konteneryzowane    | IaaS      |
| Service Fabric            | Platforma systemów rozproszonych Działa na platformie Azure lub w środowisku lokalnym                | PaaS      |
| Azure Batch               | Usługa zarządzana do obsługi przetwarzania równoległego i aplikacji obliczeniowych o wysokiej wydajności  | PaaS      |
| Cloud Services            | Usługa zarządzana do uruchamiania aplikacji w chmurze                            | PaaS      |
| Azure Container Instances | Umożliwia używanie kontenerów bez aprowizacji maszyny wirtualnej lub konieczności korzystania z usług wyższego poziomu      | FaaS      |
| Azure Functions           | Zarządzana usługa FaaS                                                      | FaaS      |

Poniżej przedstawiono funkcje sieciowe platformy Azure:

|  Nazwa usługi             | Funkcja usługi                                                                      |
| -------------             | -------------                                                                         |
| Virtual Network           | Umożliwia współdziałanie maszyn wirtualnych z przychodzącymi połączeniami wirtualnej sieci prywatnej (VPN)                     |
| Moduł równoważenia obciążenia             | Umożliwia równoważenie przychodzących i wychodzących połączeń z aplikacjami lub punktami końcowymi usług         |
| Application Gateway       | Umożliwia optymalizację dostarczania farmy serwerów aplikacji przy jednoczesnym zwiększeniu bezpieczeństwa aplikacji               |
| VPN Gateway               | Umożliwia dostęp do sieci wirtualnych platformy Azure za pośrednictwem bram VPN o wysokiej wydajności                   |
| System DNS platformy Azure                 | Zapewnia bardzo szybkie odpowiedzi DNS i bardzo wysoką dostępność domeny                   |
| Content Delivery Network  | Umożliwia globalne dostarczanie klientom zawartości wymagającej wysokiej przepustowości                                  |
| Azure DDoS Protection     | Zapewnia ochronę aplikacji hostowanych na platformie Azure przez atakami DDoS   |

| Azure Firewall            | Umożliwia implementację zapory o wysokim poziomie zabezpieczeń, wysokiej dostępności i nieograniczonej skalowalności         |
| Wirtualna sieć WAN               | Umożliwia tworzenie ujednoliconej sieci rozległej (WAN) łączącej lokalne i zdalne lokacje           |

### <a name="storage"></a>Magazyn

Platforma Azure udostępnia cztery główne typy usług magazynu. Te usługi to:

- **Azure Blob Storage** — udostępnia magazyn do przechowywania bardzo dużych obiektów, takich jak pliki wideo i mapy bitowe
- **Azure File Storage** — tworzy udziały plików, którymi można zarządzać i z których można korzystać tak jak w przypadku serwera plików
- **Azure Queue Storage** — implementuje magazyn umożliwiający kolejkowanie komunikatów oraz ich niezawodne rozsyłanie między aplikacjami
- **Azure Table Storage** — zawiera magazyn NoSLQ służący do hostowania nieustrukturyzowanych danych, niezależnych od żadnego schematu

Usługi te mają wspólne cechy:

- Niezawodność i wysoka dostępność z nadmiarowością i replikacją.
- Bezpieczeństwo dzięki automatycznemu szyfrowaniu i kontroli dostępu opartej na rolach.
- Skalowalność z praktycznie nieograniczonym magazynem.
- Funkcje zarządzania z automatyczną obsługą konserwacji i rozwiązywaniem wszystkich problemów krytycznych.
- Dostępność w dowolnej lokalizacji za pośrednictwem protokołu HTTP lub HTTPS.

### <a name="mobile"></a>Komórkowy

Platforma Azure pozwala deweloperom szybko i łatwo tworzyć atrakcyjne aplikacje dla systemów iOS, Android i Windows przy użyciu wielu języków i ulubionych środowisk projektowych. Teraz można znacznie szybciej i łatwiej dołączać funkcje, które kiedyś powodowały zwiększenie ryzyka związanego z projektem. Przykłady obejmują logowanie firmowe i łączenie się z zasobami lokalnymi, takimi jak systemy SAP, Oracle, SQL Server i SharePoint.

Oto niektóre z pozostałych funkcji tej usługi:

- Synchronizacja danych w trybie offline.
- Łączność z danymi lokalnymi.
- Emisja powiadomień push.
- Automatyczne skalowanie w celu dopasowania rozwiązania do potrzeb biznesowych.

| -------------             | -------------                                                                                   |

| Azure Database for MySQL  | W pełni zarządzana, skalowalna relacyjna baza danych MySQL o wysokiej dostępności i niezawodnych zabezpieczeniach        |
| Azure DB dla PostgreSQL   | W pełni zarządzana, skalowalna relacyjna baza danych PostgreSQL o wysokiej dostępności i niezawodnych zabezpieczeniach   |
| SQL Server na maszynach wirtualnych         | Hostowanie firmowych aplikacji programu SQL Server w chmurze                                                  |
| SQL Data Warehouse        | W pełni zarządzany magazyn danych z wbudowanymi zabezpieczeniami na każdym poziomie skalowania, które nie generują dodatkowych kosztów    |
| Azure DB Migration    | Umożliwia migrowanie baz danych do chmury bez zmian w kodzie aplikacji                            |
| Pamięć podręczna Redis               | Umożliwia buforowanie często używanych i statycznych danych w celu zmniejszenia opóźnienia przesyłania danych i opóźnienia aplikacji                    |
| Azure DB for MariaDB      | W pełni zarządzana, skalowalna relacyjna baza danych MySQL o wysokiej dostępności i niezawodnych zabezpieczeniach        |

### <a name="web"></a>Sieć Web

Poniżej wymieniono niektóre usługi internetowe na platformie Azure:

- App Service — szybkie tworzenie zaawansowanych internetowych i mobilnych aplikacji w chmurze
- Content Delivery Network —bezpieczne i niezawodne dostarczanie zawartości o szerokim zasięgu globalnym
- Notification Hubs — wysyłanie powiadomień push na każdą platformę z dowolnego zaplecza
- API Management — bezpieczne publikowanie interfejsów API dla deweloperów, partnerów i pracowników w dowolnej skali
- Azure Search — w pełni zarządzane wyszukiwanie jako usługa
- Web Apps — szybkie tworzenie i wdrażanie aplikacji internetowych o kluczowym znaczeniu w odpowiedniej skali
- Azure SignalR — łatwe dodawanie funkcji internetowych działających w czasie rzeczywistym
